Installation steps


1. Download & enable module dependencies using composer & drush.

2. To add the block to any page go to 'admin/structure/block'.

3. Search for 'Survey Invitation' block

4. Add block to the content region.

5. Click the configure button to add configurations for the block.

6. Create a basic 'news' content type by going to 'admin/structure/types'.

7. Create a node content based on the news content type or any content type.

8. View the created page and you will see the block.
