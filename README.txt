Installation steps


1. Download & enable module dependencies using composer & drush.

2. To add the block to any page go to 'admin/structure/block' and search for 'Survey Invitation' block.

3. Add block to the content region.

4. Click the configure button to add configurations for the block.

4. Create a basic 'news' content type by going to 'admin/structure/types'.

5. Create a node content based on the news content type or any content type.

6. View the created page and you will see the block.
